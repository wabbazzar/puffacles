# Puffy Queen

A simplified Killer Queen-inspired 1v1 arcade game featuring Puffy the cat.

## Game Concept

- **1v1 arena combat** with AI-controlled workers
- **Two win conditions**: Kill enemy queen 3x (Military) or collect 6 treats (Economic)
- **Tap-to-flap flying** mechanics with dive attacks

## Quick Start

```bash
make serve      # Start development server
make queen-test # Open queen movement test
```
